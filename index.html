📖 Supprimer les comptes Firebase (étapes)

Pour supprimer les comptes email/mot de passe (pas possible depuis l'app), suis ces étapes sur Firebase Console :

1
Ouvre Firebase Console
Va sur console.firebase.google.com et sélectionne ton projet lineops-production
2
Va dans Authentication
Menu gauche → Build → Authentication → onglet Users
3
Sélectionne tous les utilisateurs
Coche la case en haut du tableau pour tout sélectionner (sauf ton compte admin)
4
Supprime les comptes
Clique sur les 3 points ⋮ à droite de chaque utilisateur → Delete account
✓
Les techniciens recréent leur compte
Ils utilisent le bouton "Créer un compte" sur la page de connexion et repartent de zéro
🔗 Ouvrir directement la page Authentication →
